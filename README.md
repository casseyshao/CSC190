# CSC190 Computer Algorithms and Data Structures

**Lab 1:** Stacks and using stacks to build a parenthesis checker (Python and C).

**Lab 2:** Defining classes for the general tree and binary tree, and converting between the two (Python).

**Lab 3:** Binary Search Trees (treesort) and Self-Balancing AVL Trees (rotations) (C).

**Lab 4:** Heap traversal (inorder, preorder, postorder), adding elements to the heap, finding the existence of an element with a specified key value in the heap, and removing the max value (C). Hash function (Python).
